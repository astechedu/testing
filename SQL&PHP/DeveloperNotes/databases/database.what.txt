1. Hierarchical Database Model 
   A child has only one parent.
 
                         Root
                          |
          --------------------------------------
          |               |                    |
       level 1          level 1             level 1
        child            child               child
      |     |                                  |
 level 2    level 2                         level 2
 child      child                           child    




2. Network Data Model           
   A child has multiple parents.
   A parent has multiple childs
                                        
              
          A
       /     \                                 
      B1     B2                        
     /  \  /  \
    C1   C2  C3
